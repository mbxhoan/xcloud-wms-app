# Device Lab Checklist

## Device info

- [ ] Brand/model recorded.
- [ ] Android version recorded.
- [ ] App version recorded.
- [ ] Scanner adapter selected.
- [ ] Network type tested: Wi-Fi/4G.

## Scanner tests

- [ ] QR scan success.
- [ ] Code128 scan success.
- [ ] EAN13 scan success if needed.
- [ ] Trigger physical button works.
- [ ] Scan while screen focused works.
- [ ] Scan after app resume works.
- [ ] Scan 50 times continuous no crash.
- [ ] Duplicate debounce works.

## Operation tests

- [ ] Stock lookup.
- [ ] GR receive.
- [ ] GI pick.
- [ ] PA session.
- [ ] IC count.

## Stability

- [ ] App background/resume.
- [ ] Device sleep/wake.
- [ ] Network disconnect/reconnect.
- [ ] Battery acceptable after 2 hours.
- [ ] No overheating.
