# PDA Vendor Onboarding Checklist

Gửi checklist này cho nhà cung cấp PDA trước khi mua/chốt model.

## Questions for vendor

1. Thiết bị chạy Android version nào? Có update firmware không?
2. Scanner engine hỗ trợ barcode/QR nào?
3. Có SDK Android chính thức không? Cung cấp sample app không?
4. SDK có hỗ trợ Kotlin/Java và target SDK hiện đại không?
5. Có hỗ trợ broadcast intent khi scan thành công không?
6. Có hỗ trợ keyboard wedge không?
7. Có cấu hình suffix Enter/Tab/None không?
8. Có thể khóa thiết bị bằng MDM/kiosk mode không?
9. Có lấy được serial number/device id qua SDK không?
10. Có bảo hành/thay pin/phụ kiện dock không?

## Required materials

- [ ] SDK docs.
- [ ] Sample Android project.
- [ ] Scanner configuration manual.
- [ ] Firmware version list.
- [ ] MDM/kiosk docs if available.
- [ ] Warranty and support contact.
